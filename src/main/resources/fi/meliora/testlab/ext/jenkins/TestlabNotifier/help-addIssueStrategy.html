When to add issues:

  • Do not add issues
  • Add issues per test run: Adds a single issue per test run, and the issue's description contains details on failed tests
  • Add an issue per Testlab test case: Adds an issue for every failed Testlab test case, and the test case's description contains details on failed tests
  • Add an issue per test result: Adds an issue for every failed result

Overrides the setting for the ruleset configured in Testlab.
