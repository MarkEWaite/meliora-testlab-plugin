If you are using Testlab on your own server select this option. For using Testlab, you have two options:

  • Using hosted Testlab provided by Meliora Ltd at melioratestlab.com. For this, you should enter your company identifier which is used to construct the full url of your Testlab such as 'https://yourcompanyid.melioratestlab.com'.
  • Using Testlab on-premise on your own server. You should enter a full URL address of your Testlab server such as 'https://testlab.mycompany.com'.
